# 🧰 Tool-Box 

A personal tool-box to organize and quickly open all your online tools.

## Features

- ➕ Add new tools with Name, URL, and Category
- 🗑️ Delete tools
- 🔍 Search tools by name
- 🏷️ Filter by category (AI Tools, News Tools, Work Tools, Other)
- ↕️ Drag and reorder tools
- 💾 Auto-save using LocalStorage
- 🌙 Dark Mode / Light Mode toggle
- 📱 Responsive design (mobile + desktop)

## Default Tools

| Tool | Category |
|------|----------|
| Google Keep | Work Tools |
| Gemini | AI Tools |

## How to Use

1. Open `index.html` in your browser
2. Add your tools using the form at the top
3. Click **Open** to launch any tool in a new tab
4. Drag rows to reorder your list

## Tech Stack

- HTML
- CSS (Dark/Light theme with CSS variables)
- JavaScript (Vanilla)
- LocalStorage
